## ⚡ ACTIVER HTTP/2

### 🎯 POURQUOI ?
HTTP/2 est **2-3x plus rapide** que HTTP/1.1 pour charger les pages.

### 📋 COMMENT ACTIVER ?

#### Si hébergement OVH
1. Connexion espace client OVH
2. Hébergements → Votre domaine
3. Onglet "Informations générales"
4. Section "Configuration" → Modifier
5. Cocher "HTTP/2" → Valider

#### Si hébergement O2Switch
1. cPanel → Domaines → Gestionnaire de domaines
2. Sélectionner clesducabanon.fr
3. Activer "Force HTTPS" + "HTTP/2"

#### Si hébergement Cloudflare (CDN)
1. Dashboard Cloudflare
2. Onglet "Network"
3. HTTP/2 → ON
4. HTTP/3 (QUIC) → ON (encore plus rapide)

#### Si hébergement autre (Nginx/Apache)
**Pour Nginx** : Ajouter dans `/etc/nginx/sites-available/clesducabanon.fr`
```nginx
server {
    listen 443 ssl http2;
    server_name www.clesducabanon.fr clesducabanon.fr;
    # ... reste config
}
```

**Pour Apache** : Activer module HTTP/2
```bash
sudo a2enmod http2
```
Puis dans VirtualHost :
```apache
Protocols h2 h2c http/1.1
```
