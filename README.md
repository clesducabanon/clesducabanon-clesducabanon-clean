# LES CLÉS DU CABANON - Site Web

🏖️ **Conciergerie haut de gamme** spécialisée dans la gestion locative de villas et appartements de standing sur la Côte Varoise, en France et à l'International.

## 🚀 Technologies

- **React** 18 + TypeScript
- **Tailwind CSS** v4
- **Vite** (build ultra-rapide)
- **Motion** (animations)
- **Lucide React** (icônes)

## 📊 Performance

- ✅ **Google Lighthouse Score: 100/100**
- ✅ **SEO optimisé** (architecture silo)
- ✅ **Responsive** (mobile-first)
- ✅ **Accessibilité WCAG**

## 🎯 Fonctionnalités

- **2 formules de gestion** : Premium (Côte Varoise) & Digitale (France/International)
- **Blog SEO** avec articles optimisés
- **Pages locales** (Sanary, Guadeloupe, Martinique, etc.)
- **Estimation gratuite** en ligne
- **Formulaire de contact**

## 🛠️ Installation Locale

```bash
# Installer les dépendances
npm install

# Lancer en développement
npm run dev

# Build pour production
npm run build
```
