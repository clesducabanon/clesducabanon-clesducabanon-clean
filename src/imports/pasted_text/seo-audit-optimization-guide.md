Audit et optimise entièrement ce site pour obtenir 100/100 au score SEO sur Google Lighthouse.

Analyse le design, la structure HTML et les contenus puis corrige automatiquement tous les points suivants.

OBJECTIF :
- Score SEO : 100/100
- Performance : minimum 90
- Accessibilité : minimum 95
- Bonnes pratiques : 100

1 — STRUCTURE HTML SEO
Vérifie et corrige :

- une seule balise H1 par page
- hiérarchie correcte H1 > H2 > H3 > H4
- utilisation des balises sémantiques :
<header>
<nav>
<main>
<section>
<article>
<footer>

- toutes les images possèdent un alt descriptif contenant des mots clés pertinents
- tous les liens ont un texte d'ancrage clair

2 — META SEO

Ajoute ou optimise :

<title>Mot clé principal | Nom marque</title>

<meta name="description" content="Description SEO persuasive entre 140 et 160 caractères">

<link rel="canonical" href="URL-page">

<meta name="robots" content="index,follow">

<meta name="viewport" content="width=device-width, initial-scale=1">

Règles :
- Title entre 50 et 60 caractères
- Mot clé principal présent dans title, H1 et description

3 — OPEN GRAPH + SOCIAL SEO

Ajoute :

<meta property="og:title">
<meta property="og:description">
<meta property="og:image">
<meta property="og:url">
<meta property="og:type" content="website">

Ajoute également Twitter Cards.

4 — IMAGES SEO

Optimise toutes les images :

- format WebP ou AVIF
- taille maximale < 200kb
- lazy loading activé
- dimensions définies pour éviter CLS

<img loading="lazy">

5 — PERFORMANCE

Optimise pour Google Lighthouse :

- minification CSS
- minification JS
- suppression du JS inutilisé
- scripts chargés avec defer
- preload des images critiques (hero)
- preload des polices

Objectif Core Web Vitals :

LCP < 2.5s  
CLS < 0.1  
INP < 200ms

6 — DONNÉES STRUCTURÉES (SCHEMA)

Ajoute JSON-LD schema.org :

Organization
WebSite
Breadcrumb
Article ou Service selon la page

Format JSON-LD valide.

7 — CONTENU SEO

Optimise le contenu :

- minimum 800 mots par page principale
- mot clé principal dans :
  - H1
  - premier paragraphe
  - H2
  - meta title
  - meta description

- champ lexical riche
- texte naturel et utile

8 — MAILLAGE INTERNE

Ajoute :

- minimum 4 liens internes par page
- ancres optimisées SEO
- structure de silo logique

9 — INDEXATION

Créer ou vérifier :

robots.txt

Autoriser l'exploration sauf pages privées

Créer sitemap.xml avec toutes les pages indexables.

10 — MOBILE SEO

Optimiser le design pour mobile :

- responsive parfait
- taille texte minimum 16px
- boutons minimum 48px
- aucune barre horizontale

11 — ACCESSIBILITÉ

Ajouter :

- aria-label sur boutons et icônes
- contraste texte conforme WCAG
- navigation clavier possible

12 — ANALYSE FINALE

Après optimisation :

- simule un audit Google Lighthouse
- indique les points restant à corriger
- applique automatiquement les corrections nécessaires.

Objectif final :
Score SEO 100/100.
